## Installation
To run this project, you need to have Node.js and MongoDB installed on your system.

1. Clone this repository: `https://github.com/Yasmeen-Hezma/Tours-Reservation-API.git`
2. Install the dependencies: `npm install`
3. Start the server: `npm run start:prod`
